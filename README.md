# Create a Login Form in HTML and CSS | Glassmorphism Login Form in HTML & CSS

Um projeto simples de **formulário de login com efeito glassmorphism** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **login form moderno com estilo glassmorphism**, utilizando transparência e blur para um visual elegante.

## 🛠️ Tecnologias

- HTML
- CSS (backdrop-filter, blur)

## 🎯 Funcionalidades

- Formulário de login estilizado
- Efeito glassmorphism
- Design moderno e minimalista

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
